<h1>📌 Project Title</h1>
<b>Pro Task Manager Web App</b>

<h1>📖 Overview</h1>
The Pro Task Manager Web App is a lightweight and user-friendly task management system designed to help users efficiently organize their daily activities. This application provides essential task management features such as task creation, prioritization, due date selection, and filtering options.
<br>It is built using Vanilla JavaScript, making it a great project for understanding core frontend development concepts without relying on frameworks.<br>

<h1>🚀 Features</h1>
<li>✅ Add new tasks with custom names</li>
<li>📅 Assign due dates to tasks</li>
<li>🔢 Set task priorities (P1 - P4)</li>
<li>📂 Task filtering system:</li>
 •  Inbox (All Tasks)<br>
 •  Today’s Tasks<br>
 •  Completed Tasks<br>
<li>🧩 Simple and clean UI layout</li>
<li>⚡ Fast and lightweight (No external libraries)</li>

<h1>🛠️ Technologies Used</h1>
<li>HTML5</li>
<li>CSS3</li>
<li>JavaScript (Vanilla JS)</li>
